# KATANAZERO
《카타나 제로》는 독립 개발자 저스틴 스탠더가 개발하고 디지털 디볼버가 배급한 2019년 핵 앤드 슬래시 플랫폼 게임이다. 
macOS, 마이크로소프트 윈도우 및 닌텐도 스위치로 제작돼 2019년 4월 18일 출시됐다. 
게임 배경은 네오느와르풍의 반이상향 대도시이다. 

개발: Askiisoft

이 레포지토리는 카타나 제로를 모작하였습니다.

- 사용 엔진 : WindowAPI
- 사용 언어 : C++
- 사용 환경 : VisualStudio
- 사용 라이브러리 : FMOD(사운드)
